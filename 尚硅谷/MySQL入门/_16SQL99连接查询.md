# SQL 99 连接查询

>   语法
>
>   ```sql
>   SELECT
>   	查询列表...
>   FROM
>   	表1 AS 别名 连接类型
>   		JOIN 表2 AS 别名 
>   	ON 连接条件
>   WHERE
>   	筛选条件
>   GROUP BY
>   	分组列表
>   HAVING
>   	分组后筛选
>   ORDER BY
>   	排序;
>   ```
>
>   ### 连接类型
>
>   	* 内连接 INNER
>   	* 左外连接 LETF OUTER
>   	* 右外连接 RIGHT OUTER
>   	* 全外连接 FULL OUTER ***MySQL不支持***
>   	* 交叉连接  CROSS
