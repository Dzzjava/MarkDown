# Group

| 方法                | 传参          | 返回                 | 注释                                                         |
| ------------------- | ------------- | -------------------- | ------------------------------------------------------------ |
| getChildren         |               | ObservableList<Node> | 获得子组件容器,可使用 add,addAll 向内添加组件,同样也可以删除组件 |
| setAutoSizeChildren | boolean       |                      | 设置子组件自动管理宽高                                       |
| contains            | double,double | boolean              | 检测此位置是否有 group 的子节点                              |
|                     |               |                      |                                                              |
