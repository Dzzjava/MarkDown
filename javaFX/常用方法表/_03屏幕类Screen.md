# Screen

| 方法            | 传参 | 返回        | 注释                                     |
| --------------- | ---- | ----------- | ---------------------------------------- |
| getprimary      |      | Screen      | Screen 为单例,通过此方法获取对象         |
| getBounds       |      | Rectangle2D | 返回屏幕的宽高,通过 Rectangle2D 来获取   |
| getVisualBounds |      | Rectangle2D | 返回可视范围的宽高,通过 Rectangle2D 获取 |
| getdpi          |      | double      | 获得屏幕的dpi  / 像素/英寸               |
|                 |      |             |                                          |
